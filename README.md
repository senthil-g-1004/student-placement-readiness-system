# student-placement-readiness-system
End-to-end ML-based web application to predict student placement readiness using MySQL, Python, and Flask

## Features
- Student data stored in MySQL
- Machine learning model for placement prediction
- Flask backend for prediction service
- HTML-based user interface

## Tech Stack
- Python
- Flask
- MySQL
- Scikit-learn
- HTML/CSS

## How it Works
1. Student details are entered through a web form
2. Flask backend processes the input
3. ML model predicts placement readiness
4. Result is displayed on the UI

## Note
Database credentials are excluded for security reasons.
